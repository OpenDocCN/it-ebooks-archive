# TutorialsPoint 电子书 DockerHub 备份

## 下载

```
docker pull apachecn0/tutorialspoint-ebooks-zh
docker run -tid -p <port>:80 apachecn0/tutorialspoint-ebooks-zh
# 访问 http://localhost:{port} 查看文档
```
