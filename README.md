## 下载

### Docker

```
docker pull apachecn0/it-ebooks-archive
docker run -tid -p <port>:80 apachecn0/it-ebooks-archive
# 访问 http://localhost:{port} 查看文档
```



### NPM

```
npm install -g it-ebooks-archive
it-ebooks-archive <port>
# 访问 http://localhost:{port} 查看文档
```
